JAKARTA, KOMPAS.TV - Satu unit mobil terbakar di ruas tol MT Haryono.   <br>Tidak ada korban jiwa dalam peristiwa tersebut, api berhasil dipadamkan setengah jam kemudian dengan pengerahan satu unit mobil pemadam kebakaran.   <br>Api dengan cepat membakar mobil minibus ini.   <br>Kebakaran terjadi tepat di ruas Tol Dalam Kota Cawang, Jatinegara, Jakarta Timur.   <br>Petugas patroli jalan tol sempat memadamkan api dengan menggunakan alat pemadam ringan.   <br>Barulah setelah 10 menit kemudian satu unit mobil Damkar dari Sudin Pemadam Kebakaran Jakarta Timur tiba dilokasi dan langsung berhasiil api dipadamkan.   <br>Menurut pemilik mobil, api tiba-tiba muncul dari bagian mesin dan langsung membesar.   <br>
