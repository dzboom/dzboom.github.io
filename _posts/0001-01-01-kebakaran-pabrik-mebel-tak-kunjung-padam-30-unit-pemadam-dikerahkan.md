JAKARTA, KOMPAS.TV - Satu hektar gudang pabrik kayu, milik PT Wintrad di Jalan Raya Penggilingan, Cakung, Jakarta Timur ludes terbakar.   <br>Hingga saat ini petugas masih berusaha memadamkan api.   <br>Lebih dari tujuh jam, api tak kunjung padam.   <br>Tiga puluh unit pemadam kebakaran Suku Dinas Damkar Jakarta Timur, Jakarta Utara, dan Kota Bekasi dikerahkan, guna memadamkan kobaran api.   <br>150 personel juga ikut disiagakan.   <br>Angin kencang, material kayu yang mudah terbakar, hingga minimnya sumber air disebut jadi kendala bagi petugas untuk memadamkan api.   <br>Lokasi kebakaran berada tepat di tengah pemukiman dan juga sisi pasar.   <br>Sehingga petugas menyiapkan dua mekanisme, yakni memadamkan api dari dalam dan juga mengantisipasi dari sisi sisi gudang agar tak merambat ke pemukiman.   <br>Dari pantauan udara kamera tanpa awak, terlihat pabrik mebel yang berlokasi di jalan raya bekasi, cakung jakarta timur ini ludes terbakar.   <br>18 unit mobil pemadam kebakaran dikerahkan ke lokasi.   <br>Terlihat dari gambar udara, api menghanguskan hampir seluruh bagian pabrik.   <br>Menurut saksi mata, api pertama kali muncul di bagian gudang penyimpanan produk mebel.   <br>Untuk memadamkan kebakaran, 13 unit mobil pemadam dikerahkan ke lokasi.   <br>Petugas langsung melakukan pemadaman untuk mencegah kebakaran merambat ke permukiman warga.   <br>Menurut salah satu karyawan, api dengan cepat tiba-tiba muncul dari dalam gudang dan langsung membesar.   <br>Pabrik berisi kayu dan bahan yang mudah terbakar, membuat api cepat merambat dan membesar.    <br>Proses pemadaman terkendala karena minimnya sumber air.   <br>
