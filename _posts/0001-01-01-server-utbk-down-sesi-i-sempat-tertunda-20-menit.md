SOLO, KOMPAS.TV - Server untuk ujian tertulis berbasis komputer atau UTBK sempat mengalami gangguan, pada Minggu (5/7/2020). Akibatnya proses ujian masuk universitas sempat tertunda selama dua puluh menit.   <br>Seperti yang dialami oleh Universitas Sebelas Maret Solo, Jawa Tengah, UTBK sesi pertama harus tertunda. Gangguan pada server terjadi di tengah sejumlah calon mahasiswa sudah mulai mengerjakan soal ujian.   <br>Namun gangguan bukan hanya terjadi di UNS saja, melainkan semua lokasi. Ini dikarenakan gangguan berasal dari server pusat yang berada di universitas Padjajaran Bandung.   <br>Pihak panitia pun melakukan sejumlah langkah untuk mengatasi persoalan server ini.   <br>&quot;Karena server ini pusatnya di Universitas Padjajaran, tentu kami hanya tinggal menunggu saja setelah ada perbaikan dari sana dan rasanya tidak ada masalah yang serius,&quot; ujar Jamal Wiwoho, Rektor UNS.   <br>Namun pihak UNS menyatakan, secara keseluruhan pelaksanaan UTBK tetap berjalan dan selesai dengan baik.   <br>
