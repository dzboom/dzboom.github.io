Man Upcycles Waste To Build Sensor Activated Sanitization Machine

06 Sep 2020 by Abrega Slona
Report

This man collected discarded items from his neighbourhood and came up with an amazing sensor-activated sanitization unit. From soap dispensing to hand drying, he demonstrated how to use this unique contactless machine made with old containers, batteries and a used hair dryer.
