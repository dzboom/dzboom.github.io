Mississippi To Change Confederate State Flag To A New Design

09 Jul 2020 by Abrega Slona
Report

Tate Reeves is set to sign a bill Tuesday evening in the southern state of Mississippi.
The bill is to remove the Confederate emblem from the official Mississippi state flag.
Mississippi is the only remaining state flag to feature the Confederate insignia, reports CNN.
The Mississippi state Legislature made history on Sunday by passing a bill to retire the current flag.
They will commission a few new designs a new one to be voted on in the fall.
Confederate symbols have been contentious for decades, if not longer, and change is long wanted for some.
